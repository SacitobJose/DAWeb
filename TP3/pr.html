Resumo

Este trabalho tem como objetivo consolidar os conteúdos lecionados na aula de 01/10/2019.

Para o TPC nº3, foi pedido que os alunos gerassem, com recurso a uma stylesheet XSLT, uma página HTML para o presente project record (PR).

Foi ainda pedido que os alunos, através da mesma técnica, repetissem o exercício da aula, gerando um website para um documento XML sobre arqueossítios em Portugal.

Para isso foi criada uma página principal, com um índice para todos os arqueossítios presentes no documento, além de várias páginas secundárias, uma por cada arqueossítio. Cada entrada no índice tem uma hiperligação para a página em questão.

Documentos

  • Documento XML sobre arqueossítios;
  • Stylesheet XSLT para gerar HTML do XML acima;
  • Ficheiros correspondentes ao website dos arqueossítios;
  • Stylesheet XSLT para gerar HTML deste PR;
  • HTML gerado para este PR;
  • XML escrito para este PR.
