Resumo

Este trabalho tem como objetivo consolidar os conteúdos lecionados na aula de 22/10/2019.

Para o TPC nº6, foi pedido que os alunos codificassem um servidor em node.js capaz de manter um arquivo sonoro.

Para isso, o arquivo é mantido no servidor sob a forma de um ficheiro JSON, permitindo a consulta de toda a informação e seguindo a filosofia CRUD:

1 - criação de novos elementos para o arquivo sonoro;

2 - consulta de um elemento do arquivo sonoro;

3 - atualização de um elemento do arquivo sonoro;

4 - eliminação de elementos do arquivo sonoro;
