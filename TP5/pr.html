Resumo

Este trabalho tem como objetivo consolidar os conteúdos lecionados na aula de 15/10/2019.

Para o TPC nº5, foi pedido que os alunos codificassem um servidor em node.js capaz de manter uma lista de tarefas.

Para isso, a lista é mantida no servidor sob a forma de um ficheiro JSON, permitindo a adição de novas tarefas e a consulta das mesmas.
